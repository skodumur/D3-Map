Analysis of number of people without internet in USA

Distribution of people without internet in USA

Description:

The main idea of these visualizations is to analyze what percent of people in each state have no internet connection in USA.

The first Visualization(Chloropleth) gives an overview of the percentage of total population who don't have internet connection in each state, here the color intensity of each state signifies the percentage of people without internet.

On selecting any of the country's bar in overview chart, it populates the detailed view chart(line chart) with the selected country's performance in each year using time series line chart. Similar to the first chart this has two dimensions the green line signifies wins and orange for loses. Note: By default when overview chart is loaded, detail view chart is populated with the first country in the overview chart.

Example: On click of 'RUS' bar in overview chart the detail view shows Russia's year wise performance in detail view chart

Extra Credit: Many countries from different continents have participated in AUS open. So, displaying all countries data in a single chart might make it look clumsy. So, filtering countries based on a continent is provided on top of the overview chart. Example: When a user is interested in Asian countries performance then he needs to select Asia from a select drop-down then both the charts reflect data of Asian countries.
